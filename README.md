# Gujarat Bio Tech Pvt. Ltd. (GBT)

Corporate website for Gujarat Bio Tech — built with React 19, Vite, Tailwind CSS v4, and Framer Motion. 

## Developed Features
- Fully responsive modern design with premium aesthetics.
- Custom CSS structure with `@layer components` integration.
- Smooth animations and page transitions.
- High-conversion floating WhatsApp CTA and contact forms.
- Re-styled active states, glassmorphism UI elements, and grid layouts.

## Setup
`npm install`
`npm run dev`
